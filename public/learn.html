Each topic includes explanation, diagram, real use case, Cisco CLI examples, and troubleshooting workflow.

Learning Roadmap

  • Foundations: OSI/TCP-IP, addressing, subnetting, cabling
  • Switching: VLANs, trunks, STP, EtherChannel
  • Routing: static routes, OSPF, route filtering
  • Services: DHCP, DNS, NAT, IPv6 transition
  • Security + Ops: ACLs, AAA, monitoring, troubleshooting
  • CCNP: BGP, advanced OSPF, redistribution, QoS, MPLS, VPNs

Systematic Workflow

Plan -> Design -> Build -> Verify -> Secure -> Monitor
L1/L2: cabling, speed/duplex, VLAN, STP
L3: IP/mask, gateway, routing table
Policy: ACL/NAT order, hit counts
Services: DHCP/DNS reachability

Reference Index

  • RFC 1918 (Private IPv4)
  • RFC 2460 / 8200 (IPv6)
  • RFC 793 (TCP), RFC 768 (UDP)
  • RFC 2328 (OSPFv2), RFC 5340 (OSPFv3)
  • IEEE 802.1Q (VLAN), 802.1D/802.1w (STP/RSTP)
  • Cisco IOS Configuration Guides + Command References
