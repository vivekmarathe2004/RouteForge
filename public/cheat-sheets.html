Subnetting Formula

Hosts per subnet = 2^(32 - CIDR) - 2
Subnets added = 2^borrowed bits
Block size = 256 - mask octet
Network: IP AND Mask
Broadcast: Network OR (inverse mask)

Layered Troubleshooting Commands

L1/L2: show interfaces, show vlan brief
L2: show spanning-tree, show mac address-table
L3: show ip route, show arp, ping, traceroute
Services: show ip dhcp binding, nslookup

Troubleshooting Checklist

  • Physical: cable, speed/duplex, interface up/down
  • Layer 2: VLAN assignment, trunk allow list, STP state
  • Layer 3: IP/mask, default gateway, routing table
  • Policy: ACL/NAT order and hit counters
  • Services: DHCP lease, DNS resolution, app port reachability

Exam Mode Tips

  • Write subnet block sizes from memory
  • Check wildcard masks by inversion
  • Use show commands before debug
  • Verify requirements before applying ACLs
  • Save config after each stable milestone
